# RUNRUN

GitHub-first mobile runner game foundation.

Stack:
- React + TypeScript
- Vite
- Capacitor for mobile wrapping
- Supabase for highscores/backend data
- Vercel for final deployment

Initial architecture:
- src/ for game shell and UI
- public/ for static assets and future mobile/web icons
- supabase/ for schema and access rules
- capacitor.config.ts for mobile packaging

This repository is the initial shell for the RUNRUN prototype.
